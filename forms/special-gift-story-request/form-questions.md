# Gift Story — Special Gift Story Request Form Questions

## 1. Purpose

This file defines the questions and fields for the Special Gift Story Request form.

The form is used to collect custom gift request details in an organized way before Gift Story accepts, prices, or prepares the request.

Submitting the form does not mean the request is confirmed.

Every request must be reviewed first.

---

## 2. Form Name

## Arabic

طلب قصة خاصة

## English

Special Gift Story Request

# 19. Form Rules

## Do

* Keep the form Arabic-first.
* Make the form clear and premium.
* Use “review” language, not instant acceptance.
* Keep WhatsApp secondary.
* Ask for enough details to evaluate the request.
* Make budget a guide, not a promise.
* Add consent checkboxes.
* Use a warm success message.

## Avoid

* Saying the request is confirmed after submission.
* Promising same-day custom gifts.
* Making every field required.
* Asking too many unnecessary questions.
* Using discount language.
* Using WhatsApp as the main form replacement.
* Showing fixed custom prices.
* Making the form feel cold or complicated.
