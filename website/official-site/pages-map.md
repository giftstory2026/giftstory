# Gift Story — Official Website Pages Map

## 1. Purpose

This file defines the page structure for the official Gift Story website.

The official website is the brand home.

It should build trust, explain the story, show the packaging experience, introduce the active category, and guide customers to the active commerce platform.

The website should not act as the main checkout system.

---

## 2. Website Role

The official website should be used for:

* Brand storytelling.
* Premium visual presentation.
* Category introduction.
* Packaging explanation.
* Previous / sample stories.
* Special Gift Story Request.
* Policies.
* Contact.
* Redirecting customers to the active shop.

The commerce platform should handle:

* Prices.
* Products.
* Inventory.
* Checkout.
* Payment.
* Orders.

---

## 3. Main Website CTA

Main CTA:

**تسوّقي القصة الحالية**

English support:

**Shop The Current Story**

CTA behavior:

* Directs customer to the active commerce platform.
* Can link to `shop.gift-story.com` or another approved platform URL.
* Should appear clearly in header, homepage, Little Stories page, footer, and mobile view.

Do:

* Keep the page elegant and spacious.

* Using discount language.

* Using raw product photos.

Show previous, sample, or inspiration gift stories without promising that the same exact boxes are still available.

## Suggested Sections

1. Page intro.
2. Gallery of previous stories.
3. Sample story cards.
4. Limited availability note.
5. Link to Special Gift Story Request.
6. CTA to current story.

## Arabic Note

بعض القصص السابقة كانت محدودة وقد لا تتكرر بنفس التفاصيل، لكن يمكن استخدامها كمرجع للإلهام أو الطلبات الخاصة حسب توفر القطع.

## Rules

Do:

* Use polished photos only.
* Show variety without confusing customers.
* Mark unavailable concepts as previous stories.
* Link to custom request page.

Avoid:

* Making old boxes look available.
* Showing prices.
* Showing raw inventory photos.
* Showing purchase sources or tags.

## Path

```text
/special-request
```

## Page Name

**طلب قصة خاصة**

English support:

**Special Gift Story Request**

## Purpose

Allow customers to request a custom gift story.

## Main Message

Some gifts need a more personal story.
This page allows customers to request a special concept based on occasion, recipient, budget, and timing.

## Arabic Intro

للمناسبات التي تحتاج تفاصيل أكثر خصوصية، يمكنك إرسال طلب قصة خاصة، وسنراجع الفكرة حسب توفر القطع المناسبة وإمكانية التنفيذ.

## Required Form Fields

```text
Full Name
Email
Phone
City
Occasion
Recipient Type
Recipient Age / Size if relevant
Preferred Style
Budget Range
Desired Delivery Date
Special Notes
Consent to be contacted
```

## Customer Note

الطلبات الخاصة تحتاج وقتًا إضافيًا للتجهيز، وتعتمد على توفر القطع المناسبة وإمكانية تنفيذ الفكرة.

## Form Status Options

```text
Active
Campaign Active
Paused
```

## Paused Message

طلبات القصص الخاصة غير متاحة حاليًا. نعمل على تجهيز قصص جديدة، ويمكنك متابعة الإصدارات القادمة قريبًا.

# 17. Website Visual Rules

The website should feel:

* Premium.
* Soft.
* Warm.
* Calm.
* Spacious.
* Elegant.
* Feminine but not childish.
* Gift-focused.
* Story-driven.

Approved visual direction:

* Off-white.
* Creamy beige.
* Nude.
* Muted / satin gold.
* Soft digital pink only when needed.
* Castle.
* Stars.
* Moon.
* Elegant packaging.

Avoid:

* Dark navy as the main identity.
* Loud colors.
* Heavy discount banners.
* Crowded grids.
* Too many icons.
* Random product photos.
* Cheap commercial visuals.

---

# 18. Website Image Rules

Use:

* Real packaging photos when available.
* Clean product photos.
* Soft mockups.
* Flat lay images.
* Detail shots.
* Category visuals.
* Previous story visuals.

Do not use:

* Raw supplier photos.
* Price tags.
* Discount tags.
* Store labels.
* Ross / MSRP.
* Dollar signs.
* Messy backgrounds.
* Low-quality screenshots.
* Personal information.

---

# 19. Website Success Criteria

The website is successful if:

* Customers understand Gift Story quickly.
* The brand feels premium.
* Little Stories is clear.
* Packaging value is visible.
* The shop button is easy to find.
* Special requests are easy to submit.
* Policies are clear.
* Mobile experience is smooth.
* No conflicting prices appear.
* The website supports sales without becoming a cheap storefront.

---

# 20. Final Website Direction

Gift Story’s official website should be a premium brand website.

It should explain the story, show the world, and guide customers to the active shop.

The website builds trust.

The commerce platform completes the order.
