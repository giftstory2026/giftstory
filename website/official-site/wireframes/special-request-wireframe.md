# Gift Story — Special Request Page Wireframe

## 1. Purpose

This file defines the visual and structural wireframe for the Special Gift Story Request page.

The page should allow customers to request a custom gift story in a premium, organized, and controlled way.

The page should not feel like instant chat ordering.

The page should make it clear that every special request is reviewed before acceptance.

---

## 2. Page Goal

The Special Request page should answer:

1. What is a special gift story?
2. When should the customer use this page?
3. What information does Gift Story need?
4. Is the request accepted immediately?
5. How does the review process work?
6. How is the price confirmed?
7. How does the customer submit the request?
8. What happens after submission?

---

## Section Title

## Steps

### Step 1

إرسال الطلب
تعبئين نموذج طلب القصة الخاصة بالمناسبة، الفئة، الميزانية، والموعد المتوقع.

### Step 2

مراجعة الفكرة
نراجع الطلب حسب توفر القطع، الوقت، وطبيعة المناسبة.

### Step 3

اقتراح التصور
إذا كان الطلب مناسبًا، نرسل لكِ تصورًا أو نطاقًا مقترحًا للهدية والسعر.

### Step 4

تأكيد الطلب
بعد موافقتكِ، يتم تأكيد الطلب والدفع عبر وسيلة رسمية.

### Step 5

التجهيز والتسليم
نبدأ تجهيز القصة بعناية، ثم يتم تسليمها أو شحنها حسب التفاصيل المعتمدة.

## Design Notes

* Use numbered steps.
* Keep each step short.
* Avoid operational complexity.
* Make the process feel premium and reassuring.

# 21. FAQ Section

## Purpose

Answer common objections and reduce confusion.

## Layout

```text id="qx4krg"
[FAQ Accordion]
```

Mobile:

```text id="zd2yyf"
[Collapsed accordion]
```

## Questions

### Question 1

هل يعني إرسال الطلب أنه تم قبوله؟

### Answer 1

لا. إرسال الطلب يعني أننا استلمنا التفاصيل للمراجعة. يتم قبول الطلب فقط بعد مراجعة الفكرة، توفر القطع، الوقت المناسب، وتأكيد السعر.

---

### Question 2

هل يمكن تنفيذ نفس قصة سابقة؟

### Answer 2

يمكن استخدام القصص السابقة كمصدر إلهام، لكن لا نضمن تكرار نفس التفاصيل لأن بعض القطع تكون محدودة أو غير متوفرة.

---

### Question 3

كم يستغرق الطلب الخاص؟

### Answer 3

يعتمد الوقت على طبيعة الفكرة، توفر القطع، وموعد التسليم المطلوب. الطلبات الخاصة تحتاج وقتًا إضافيًا مقارنة بالقصص الجاهزة.

---

### Question 4

هل يمكن تحديد ميزانية معينة؟

### Answer 4

نعم، يمكن تحديد نطاق الميزانية في النموذج، لكن قبول الطلب يعتمد على إمكانية تنفيذ هدية مناسبة ضمن هذا النطاق وبجودة تليق بـ Gift Story.

---

### Question 5

هل يمكن طلب هدية عاجلة؟

### Answer 5

قد لا تكون الطلبات الخاصة العاجلة ممكنة دائمًا، لأنها تحتاج مراجعة وتجهيز. يمكن استكشاف القصص الحالية إذا كانت الهدية مطلوبة خلال وقت قصير.

## Design Notes

* Keep FAQ answers short.
* Use accordion for mobile.
* Avoid repeating the entire policy page.

# 26. Mobile Priority

The Special Request page should be mobile-first.

Mobile priorities:

* Form fields are easy to fill.
* Labels are clear.
* Form is not too long visually.
* CTA to form appears early.
* Important note appears before form.
* FAQ is collapsed.
* Page loads fast.
* No distracting visuals.
* No WhatsApp-first flow.

---

# 27. Page Design Rules

## Do

* Make the request process feel premium and organized.
* Use the form as the main intake.
* Explain that requests are reviewed.
* Clarify time, availability, and feasibility.
* Keep the form easy to use.
* Link ready shoppers to the current story.
* Use calm note boxes and soft visuals.

## Avoid

* Making requests look instantly accepted.
* Promising exact recreation of old boxes.
* Showing fixed custom prices.
* Using WhatsApp as the main CTA.
* Making the form too complicated.
* Using urgent or discount language.
* Accepting every idea automatically.
* Making the page feel like a support ticket.

---

# 28. Final Wireframe Decision

The Special Request page should guide the customer from:

```text id="c4tyzi"
Custom gift idea → Clear expectations → Request process → Form submission → Review
```

The customer should leave this page understanding:

* Special requests are possible.
* Requests are reviewed before acceptance.
* Execution depends on availability and time.
* Price is confirmed after review.
* The form is the correct starting point.
* Ready gifts are available through the official shop.
